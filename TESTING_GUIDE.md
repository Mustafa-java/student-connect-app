## 🐛 Если возникают ошибки:

### "Register error: type 'Null' is not a subtype..."
**Решение:** Уже исправлено в `api_service.dart`. Пересоберите приложение:
```bash
flutter clean
flutter pub get
flutter run
```

### "Connection refused" / "Network error"
**Причины:**
1. Backend сервер спит (первый запрос занимает 30-50 секунд)
2. Нет интернета на устройстве
3. Неправильный URL в `api_service.dart`

**Решение:**
- Подождите 1 минуту и попробуйте снова
- Проверьте интернет на устройстве
- Убедитесь, что в `api_service.dart` строка 14:
  ```dart
  static const String _baseUrl = 'https://student-connect-backend.onrender.com';
  ```

### "Invalid token" / "Unauthorized"
**Решение:**
- Выйдите из аккаунта
- Войдите заново
- Токен обновится автоматически
