# Student Connect App

Мобильная платформа для студентов — смесь соцсети, портфолио проектов и мессенджера.

### Описание
Приложение позволяет студентам:
- Просматривать ленту проектов и идей других студентов
- Создавать и показывать своё портфолио
- Находить единомышленников по навыкам и интересам
- Общаться в чате
- Следить за событиями (хакатоны, конкурсы и т.д.)

Дизайн максимально приближен к Instagram (расстановка элементов, навигация, карточки, анимации).

**Текущая версия:** Только Android + Русский язык  
**Стек:** Flutter (frontend), NestJS + PostgreSQL (backend — в разработке)
